Stargazer

The perpetual progress machine for computational biology

Stargazer is an open-science initiative to broaden access to bioinformatics tools, foster collaboration, and align incentives in service of biomedical progress.

About

Stargazer lowers the barrier to entry by placing an agent between the researcher and a set of proven workflows made of deeply composable tasks. Executions run on declarative infrastructure against content-addressable data — reproducible by construction and ready to build on at any point in the future.

Interface

A familiar chat frontend where researchers express intent. An LLM with access to a registry of typed Flyte tasks and workflows composes — or builds — the right module for the job.

Orchestrator

Flyte V2's pure-Python, async-native engine. Every task declares its resources and dependencies inline, runs in a container, and enforces types at its boundary.

Storage

Every file is content-addressable on IPFS and self-describing via Pinata. Workflows reference data by attributes, not location, so reproducibility is intrinsic.
